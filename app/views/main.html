Add Mobbr support to your collaboration site
Plugin

The easiest way to start hosting tasks is to install collaboration software for which Mobbr has already built a plugin. Mobbr uses the software's reputation system as the basis for distributing money. Ask your system administrator to do this.

Find a plugin
HTML metadata

The most flexible way to host tasks is to have your web-designer generate a simple script into the HTML of your site, which tells us how to distribute the money if a payment is done to it. Upon payment Mobbr will do a callback to your page and find the script. You can make the distribution key as simple or as complex as you like.

Metadata integration
Connect your API to Mobbr

If your site has an API in which Mobbr can find the contribution statistics for your projects, we can integrate your site from our side. We already did this with Github, Gitlab and Stack Exchange. Please mail us if you would like your site to be connected.

API integration
 
Add Mobbr support, the quick fixes

Create this /PARTICIPATION.TXT in the root of your domain

[
    {
        "url-pattern": "*",
        "script":
        {
            "participants":
            [
                {
                    "id": "mailto:your@email.com"
                    "role": "owner",
                    "share": "1"
                }
            ]
        }
    }
]

...and/or put this <metatag> in your HTML page(s)

<html>
    <head>
    ...
    <meta name="participation" content='{
        "participants" :
        [
            {
                "id": "mailto:your@email.com",
                "role": "owner",
                "share": "1"
            }
        ]
        }'/>
    ...
    <head>
    <body>

Next enter your URL in the box above and see what happens! Use to incrementally extend the Mobbr support on your site.
